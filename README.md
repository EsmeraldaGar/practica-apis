# Practica de APIs
 
Este proyecto es una práctica para aprender a consumir y crear APIs.
 
## Objetivos
 
- Configurar un entorno de desarrollo con Python y Git.
- Implementar peticiones HTTP a APIs externas.
- Crear una API sencilla.
 
## Instalación para colaboradores
 
1. Clonar el repositorio.
2. Crear y activar entorno virtual.
3. Instalar dependencias: `pip install -r requirements.txt`
4. Copiar `.env.example` a `.env` y poner tu propia API key de OpenWeatherMap.
5. Ejecutar el servidor: `uvicorn main:app --reload`
