# Practica de APIs
 
Este proyecto es una práctica para aprender a consumir y crear APIs.
 
## Objetivos
 
- Configurar un entorno de desarrollo con Python y Git.
- Implementar peticiones HTTP a APIs externas.
- Crear una API sencilla.
 
## Instrucciones
 
1. Clonar el repositorio.
2. Crear y activar el entorno virtual.
3. Instalar dependencias.
4. Ejecutar el código.
